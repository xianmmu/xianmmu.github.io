Introduction

Web browsers are software applications that enable users to access and interact with web pages on the internet. Different web browsers have different features, capabilities, and performance levels that may affect the user experience and satisfaction.

In this document, we will explore the unique and special features of the web browsers (Chrome, Opera GX, Edge, Firefox) by using various sources of information such as official websites, reviews, benchmarks, and statistics to support our analysis.

In addition, we'll also research about and test out the customizability and efficiency of the web browsers (Chrome, Opera GX, Edge, Firefox) by using various testing options provided by browserbench.org to test different parts of the browsing experience, such as running heavy Javascript or graphics-intensive websites, as well as for consistency in variables.
